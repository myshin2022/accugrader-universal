## How It Works

### Multiple Choice (FREE)

1. Enter correct answers in `MC_AnswerKey` sheet
2. Enter student answers in `MC_Submissions` sheet
3. Click: `AccuGrader > Multiple Choice > Grade ALL students`
4. Done! Scores and feedback appear automatically

### Essay Grading

1. Get free API key from [platform.openai.com](https://platform.openai.com/api-keys)
2. Add key to Script Properties (see Instructions sheet)
3. Paste essays in `Essay_Submissions` sheet
4. Click: `AccuGrader > Essay > Grade ALL students`
5. AI grades and provides detailed feedback
